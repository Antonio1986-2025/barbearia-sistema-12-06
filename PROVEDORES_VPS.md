## 🚀 Como Contratar

### Contabo (Recomendado):

1. Acesse: https://contabo.com/en/vps/
2. Escolha: **Cloud VPS S** (€4.99/mês)
3. Sistema: **Ubuntu 22.04 LTS**
4. Localização: **Nuremberg, Germany**
5. Período: Mensal (para testar)
6. Finalize e aguarde email com credenciais

### DigitalOcean:

1. Acesse: https://www.digitalocean.com/
2. Crie conta (ganhe $200 crédito)
3. Create → Droplets
4. Escolha:
   - **Image:** Ubuntu 22.04 LTS
   - **Plan:** Basic ($12/mês - 2GB RAM)
   - **Datacenter:** São Paulo
   - **Authentication:** SSH Key (recomendado)
5. Create Droplet

### Vultr:

1. Acesse: https://www.vultr.com/
2. Crie conta (ganhe $100 crédito)
3. Deploy New Server
4. Escolha:
   - **Type:** Cloud Compute
   - **Location:** São Paulo
   - **Server:** Ubuntu 22.04 LTS
   - **Plan:** $12/mês (2GB RAM)
5. Deploy Now

## 📝 Configuração Inicial

Após contratar qualquer provedor:

1. **Anote as credenciais:**
   - IP do servidor
   - Senha root (ou chave SSH)

2. **Conecte via SSH:**
   ```bash
   ssh root@SEU_IP
   ```

3. **Siga o checklist de deploy:**
   - Veja `CHECKLIST_DEPLOY.md`
   - Tempo total: ~1 hora

## 💡 Dicas Importantes

### Escolhendo Localização:
- **Usuários no Brasil:** Escolha São Paulo
- **Performance > Latência:** Alemanha é OK (40-80ms)
- **Custo importante:** Europa (Contabo, Hetzner)

### Sobre RAM:
- **1GB:** Funciona mas fica justo
- **2GB:** Confortável (recomendado)
- **4GB+:** Sobra para crescer

### Sobre Tráfego:
- **Sistema usa:** ~50GB/mês (estimado)
- **1TB/mês:** Mais que suficiente
- **Ilimitado:** Não se preocupe

### Backup:
- Configure backup automático (geralmente $1-2/mês)
- Vale MUITO a pena

### Snapshot:
- Tire snapshot antes de mudanças grandes
- Facilita reverter se algo der errado

## 🔐 Segurança

### Após Contratar:

1. **Mude senha root** (se vier com senha)
2. **Crie usuário não-root** (barbearia)
3. **Configure firewall** (ufw)
4. **Use SSH Key** (em vez de senha)
5. **Configure Fail2Ban** (proteção força bruta)

Tudo isso está no guia de deploy!
